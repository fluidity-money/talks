Pros Cons
  • More developed tooling (Geth, easier smart contract scripting, etc)
  • Fanout broadcast of ordered data is easy and can be byzantine fault tolerant
  • Most of the time, inherit the performance restrictions of layer 1 (it's a blockchain, what'd you think?)
  • In some configurations, implies the economics to build a native token, so need raise, need inherent value
  • Less privacy than a state channel
